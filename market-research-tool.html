Market Research Tool

One keyword, nine federal and identity sources, all at once. Type a vendor, product, or capability term and get spend history, active opportunities, entity registrations, sanctions, patents, regulations, and news in parallel. Generate a market research memo when you are done.

Home / Tools / Market Research Tool
Proxy not configured. Set PROXY_BASE at the top of the script block in this page to your Cloudflare Worker URL. See market-research-worker/README.md for setup.
Type a term and hit Search. Pull as much or as little as you have. The tool fans out in parallel.

How this works. The page calls a small Cloudflare Worker proxy that holds the API keys for SAM.gov, NewsAPI, and OpenCorporates. The proxy hits all nine sources in parallel and normalizes them to one shape. Results stream back as each source responds. The memo button assembles whatever loaded into a Word doc you can drop in the contract file. Source code on GitHub.
